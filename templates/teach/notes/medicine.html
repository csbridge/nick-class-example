This is one section problem for the day. The main learning goal is using variables and concatenating different variable types together. When transitioning to this problem, take a minute to emphasize the societal impact of this solution! It turned out to save countless lives, and is a great example of Computer Science being used as a tool to solve larger problems. Here is the main way to understand how to approach the problem:

  • Initially, we may think to just loop NUM_PACKAGES times and each time print out a random number, but we may print out the same random number more than once. We need a way to print out numbers that have not been printed out before.
  • One approach is to focus on the two requirements: being unique and being random. Approach each one at a time and discuss how we might be able to satisfy it.
  • For example, one way to do this is to make just part of the number completely random. In other words, let's make the first five digits a fixed number that we increment each time we make a medicine number. Then, let's make the last five digits a random number between 10000 and 99999.
  • This will make a number 10 digits long in total, where no two numbers will be the same. This is because the first 5 digits are always unique. This also makes a number that is random, since the last 5 digits are always random.

Solution

public class Medicine extends ConsoleProgram {
  
    // The number of medicine numbers to generate
    private static final int NUM_PACKAGES = 1000;

    private RandomGenerator rgen = new RandomGenerator();
  
    public void run() {
        int firstFiveDigits = 10000;
        for (int i = 0; i < NUM_PACKAGES; i++) {
            println("" + firstFiveDigits + rgen.nextInt(10000, 99999));
            firstFiveDigits = firstFiveDigits + 1;
        }
    }

}
