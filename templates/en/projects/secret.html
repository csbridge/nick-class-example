Secret

Written by Chris

Example: Snow
Handouts: ArrayLists

Your Job

You are given a function called loadSecret() which returns a secret message as an ArrayList<String>. Each String in the ArrayList is a word in the message.

Print out each of the stings in the array list (in order) to see the secret message.

Review

In order to iterate over an ArrayList, use a for loop. For example this code calculates the sum of all the values in an ArrayList of Doubles.

ArrayList<Double> values = getValues();
int sum = 0;
for(int i = 0; i < values.size(); i++) {
   sum = sum + values.get(i);
}
println(sum);
