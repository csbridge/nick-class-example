MadLibs

Problem by Julia Lee and Bryce Cronkite-Ratcliff

Example: Funky Functions

Write a console program that prints 10 3-word sentences, each of the form "{subject} {verb} {object}." We have provided a sets of words to randomly select from (see below), but you may also create your own word sets! Here's an example run:

Your solution should include three functions (in additon to run), each of which returns a random word from the word set. You can do this by randomly picking a number 1, 2, or 3, and returning the corresponding String in that category:

  • getSubject: returns a random String from the list of subjects
  • getVerb: returns a random String from the list of verbs
  • getObject: returns a random String from the list of objects
Finally, you should concatenate and print the values returned by your functions, adding the appropriate spacing and punctuation.
